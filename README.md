# Bella Motor

Aplikasi Kasir dan Manajemen Stok Sparepart Motor

## Fitur

- Login & Register
- Dashboard
- Kasir
- Cetak Struk
- Manajemen Stok
- Stok Masuk
- Riwayat Transaksi
- Laporan Penjualan
- Export PDF
- Backup Database

## Teknologi

- HTML
- CSS
- JavaScript
- Node.js
- Express.js
- MySQL

## Cara Menjalankan

1. Import database bella_motor.sql
2. Masuk ke folder backend
3. Jalankan:

npm install
npm start

4. Buka frontend/pages/Login/login.html
